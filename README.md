# Como Fazer o Deploy de uma API na Nuvem na Prática

Projeto demonstrando o processo de deploy de uma API na nuvem utilizando integração com GitHub.

## Objetivo

Apresentar de forma prática como publicar uma API na nuvem utilizando repositório GitHub e integração com plataforma de hospedagem cloud.

## Etapas do Deploy

1. Criar a API localmente.
2. Criar repositório no GitHub.
3. Subir o código para o repositório.
4. Conectar o repositório a uma plataforma de cloud.
5. Realizar deploy automático via integração contínua (CI/CD).

## Conceitos Aplicados

- API REST
- Versionamento com GitHub
- Deploy automático (CI/CD)
- Cloud Computing
- Hospedagem em nuvem

## Conclusão

O projeto demonstra o fluxo completo de publicação de uma API na nuvem utilizando GitHub como ferramenta central de versionamento e integração com serviços cloud.
